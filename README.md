## Iniciando

Esse projeto tem como objetivo fins de estudo e aperfeiçoamento em TDD 

### Pré-requisitos

Necessário ter [NodeJS](https://nodejs.org/en/) instalado no mínimo na versão LTS

### Instalando

Na pasta base do projeto instalar as dependências com:

```
npm install
```
